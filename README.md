# 🎬 Movie App

A simple movie discovery web application built with React that allows users to browse trending movies, and manage a personal favorites list.

🔗 Live Demo: https://testmovieapptest.netlify.app/

---

## ✨ Features

- Browse trending and popular movies
- Search movies by title
- Add and remove movies from Favorites
- Favorites persistence using localStorage
- Client-side routing
- Responsive UI for mobile and desktop
- Loading and error states handling

---

## 🛠 Tech Stack

- React
- React Router
- TailwindCSS
- Fetch API
- Context API (State Management)
- TMDB API
  
## 🎯 Project Purpose

This project was built to brush up on my understanding of building real-world React applications that interact with external APIs.
The goal was to practice handling asynchronous data, implementing client-side routing, managing global state, and designing responsive user interfaces while following clean component architecture.
